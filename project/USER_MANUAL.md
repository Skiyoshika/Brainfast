# IdleBrain (MVP) 用户使用说明书

欢迎使用 IdleBrain 脑图谱细胞计数工具！这是一份专为初学者编写的指南，帮助您从零开始，轻松完成小鼠脑切片的细胞检测与图谱对齐。

---

## 🚀 1. 核心概念与准备工作

本工具的核心目标是：**将您的真实显微镜切片（Real Slice），与标准的 Allen 小鼠脑图谱（Allen Mouse Brain Atlas）对齐，并自动统计各个脑区内的细胞数量。**

### 您需要准备的数据：
1. **真实切片图像文件夹**：包含您拍摄的脑切片 TIFF 图像。
2. **单张真实切片（用于预览）**：挑选一张具有代表性的切片图像。
3. **Allen 图谱标注文件**：通常为 `annotation_25.nii.gz`（NIfTI 格式，体素间距 25µm）。
4. **单张图谱切片（用于预览）**：与您的真实切片位置相对应的图谱切片 TIFF 图像。
5. **脑区映射表**：工具自带的 `1_adult_mouse_brain_graph_mapping.csv`，用于将图谱 ID 转换为脑区名称。

> **💡 重要提示：物理缩放机制**
> 在最新的版本中，系统会自动处理图谱（25µm/像素）与真实切片（通常为 0.65µm/像素）之间的**物理尺寸差异**。系统会自动等比例缩放图谱，并使用边缘填充（Padding）或裁剪（Cropping）来对齐您的真实图像，**绝不会破坏图谱的原始纵横比**。

## 💻 2. 启动软件

您可以选择以下两种方式之一启动工具：

**方式一：使用桌面快捷方式（推荐）**
双击运行 `frontend/dist/IdleBrainUI.exe`。

**方式二：使用命令行（开发者模式）**
打开终端（终端或命令提示符），输入以下命令：
```bash
cd D:\IdleBrain\project\frontend
python server.py
```
然后，在浏览器中打开：`http://127.0.0.1:8787`

---

## ⚙️ 3. 填写文件路径

在软件主界面，您需要依次填入第一步准备好的文件路径：

- **Input TIFF Folder**：真实切片图像所在的文件夹路径。
- **Atlas annotation_25.nii.gz**：Allen 图谱标注文件的路径。
- **Structure Mapping CSV**：脑区映射表的路径（填写 `1_adult_mouse_brain_graph_mapping.csv`）。
- **Real Slice Path**：用于预览和对齐测试的单张真实切片路径。
- **Atlas Label Slice Path**：用于预览和对齐测试的单张图谱切片路径。

---

## 👁️ 4. 预览叠加效果

在进行 AI 对齐之前，我们可以先看看图谱和真实切片初步叠在一起是什么样子。

1. **设置透明度 (Overlay Alpha)**：建议设置为 `45`，这样既能看清细胞，也能看清图谱色块。
2. **选择显示模式 (Overlay Mode)**：
   - **`Fill`（填充模式）**：整个脑区会被涂上颜色，适合查看整体分布。
   - **`Contour`（轮廓模式）**：只显示脑区的边界线，适合查看边界对齐情况。
     *(注：如果在 Fill 模式下出现异常，系统会自动安全回退到 Contour 模式)*
3. 点击 **Refresh Preview**（刷新预览）按钮。
4. 此时，您可以在 `outputs/overlay_preview.png` 中查看初步的叠加效果。

---

## 🤖 5. AI 智能对齐 (Alignment)

这是最关键的一步，AI 会自动寻找图像中的特征点（Landmarks），将图谱精准地“拉伸”或“扭曲”以贴合您的真实脑切片。

1. **选择对齐模式 (Align Mode)**：
   - **`Affine`（仿射变换）**：仅进行平移、旋转、缩放和倾斜。适合切片变形较小的情况。
   - **`Nonlinear`（非线性变换）**：允许图谱发生局部扭曲。适合切片有明显变形、撕裂的情况。
2. **高级参数（可选调整）**：
   - `Max Landmarks`（最大特征点数）：默认 30~60。
   - `Min Distance`（最小特征点间距）：默认 8~15。
   - `RANSAC Residual`（容错率）：默认 6~10。
3. 点击 **AI Landmark Align**（AI 特征点对齐）按钮。
4. **查看对齐质量**：
   - 关注日志中的 `edge SSIM` 分数（结构相似度）。如果分数**提升**（after > before），说明对齐成功！
   - 点击 **View Landmarks** 可以查看特征点分布图。
   - 如果对齐效果不佳，可以尝试调低 `RANSAC Residual`，或者增加 `Max Landmarks`，然后重试。

---

## 🏃 6. 运行完整分析流水线 (Pipeline)

对齐满意后，就可以开始批量处理所有切片并统计细胞了。

1. **选择通道**：根据您的荧光染色情况，选择红光（Red）、绿光（Green）、远红光（Farred），或者直接选择 **Run All Channels**（运行所有通道）。
2. **设置质量阈值 (Quality Threshold)**：系统内置了质量门限防护，您可以配置可接受的最低质量分数。低于此分数的切片将被记录在异常报告中。
3. 点击 **Run Pipeline**（运行流水线）按钮。
4. 进度条开始滚动，请耐心等待。（如果发现设置错误，随时可以点击 **Cancel** 停止）。

---

## 📊 7. 查看结果报告

运行完成后，所有结果都会保存在项目的 `outputs/` 文件夹下。您最需要关注的是以下几个文件：

- 📍 **细胞坐标表**：`cells_detected.csv`（原始检测结果）和 `cells_dedup.csv`（去重后的结果）。
- 🧠 **脑区映射表**：`cells_mapped.csv`（每个细胞所属的图谱脑区）。
- 📈 **最终统计报告**：
  - `cell_counts_leaf.csv`：最细分脑区的细胞数量统计。
  - `cell_counts_hierarchy.csv`：按层级汇总的细胞数量统计（例如：整个皮层包含多少细胞）。
- 🛡️ **质量控制报告**：`slice_qc.csv`（记录了每张切片的处理质量，方便您排查有问题的切片）。

## 🆘 8. 常见问题排查 (Troubleshooting)

- **Q: 提示 "Path validation failed"（路径验证失败）怎么办？**
  - A: 请仔细检查您在第 3 步中填写的路径是否完全正确，确保文件和文件夹真实存在。
- **Q: 预览出来的图谱完全失真、被严重压扁了？**
  - A: 这个在最新版（包含物理缩放修正）中已经解决。请确保您使用的是最新版本的代码，AI 会自动读取 25µm 的体素间距并与真实图像匹配，保持原始纵横比。
- **Q: 运行过程中某个环节失败了，需要重头再来吗？**
  - A: 不需要！我们的系统支持“一键重试”。修复问题后再次点击运行，系统会自动从上次断开的地方继续，节省您的时间。
